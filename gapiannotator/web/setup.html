Welcome to the gapi-annotator!
This application allows you to automatically tag your images using the Google API and store the tags in the metadata of the images. It offers the following features:
  • Tagging based on the image content: cloud, water, animals, cars, wheels, Eifel tower, company logos, dominant colors, ...
  • Optical Character Recognition (OCR) on text within the image, will also be stored as a label.
  • Reverse geocoding: add location (town, region, country) if the image has a GPS tag.
  • All labels can be translated to a desired language.
  • Face detection, whereby the faces can easily be named using the Web GUI.
  • Lossless rotation of images according to their EXIF orientation tag.
  • While running, given folder will continuously be watched for new images to automatically process them.
  • If running on a Synolgy NAS: automatically create synolgy thumbnails, faster than the buildin method. All changes will automatically pushed to the indexing service.
  • Automatic synchronization between simultaneously used Web GUI instances using websockets. This allows, for example, that multiple users can name faces in parallel.
  • The Web GUI is optimized for desktop as well as mobile usage.

Requirements
First, you need a Google Developer account and have to create a service account (credentials file) as well as an API key (stored in a file).
Furthermore you need to activate the follwing APIs: Cloud Vision, Cloud Translation, and Geocoding.
Google APIs have free contingent per month, which should be enough for most users. Here you can find pricing details for the used APIs: Vision, Translation, and Geocoding.
Note: to validate the API key, the credentials file, and that the required APIs are activated, a test request will be sent for each.
